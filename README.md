# homebrew-gitagotchi

Homebrew tap for [gitagotchi](https://github.com/wjames111/gitagotchi) — a
terminal Tamagotchi whose entire life is derived from your GitHub account.

```sh
brew install wjames111/gitagotchi/gitagotchi
gh-pet          # your pet
```

The formula is bumped automatically by the gitagotchi release workflow on
every tagged release.
